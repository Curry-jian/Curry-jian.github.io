linux基础

基本命令：
hostname : 查看主机名
hostname 主机名： 临时修改主机名
su - 用户名 ： 切换用户登陆系统
useradd 用户名： 增加一个用户
pwd : 查看当前的在哪个文件夹
cd : 进入文件夹
cd ../../ 返回到上一级的上一级目录
cd 用户： 进入某个用户的家目录
cd - : 返回到上次工作的目录
ls: 查看文件夹的内容
-l(long format):显示详细信息 -a(all)：显示所有的文件夹和文件包括隐藏文件
-R(recurisive) :递归显示 -d(directory) 文件名: 查看某个文件夹下某个文件的属性
-h（human-readable）:人类可读大小的形式
mkdir: 新建文件夹
mkdir .文件名 建立隐藏文件夹
-p : 当父文件夹不存在的时候，就新建父文件夹 mkdir -p changsha/{yuelushang，hu}
mkdir lijian{1..10}: 代表建立10个李坚 序号由110
echo :查看某个文档或则变量
man bash:查看bash命令的使用方法和手册 / 进行查找 n 查找下一个
reboot： 重启系统
file : 查看文件的类型
echo $SHELL : 查看Linux里面默认的shell
which : 查看命令存放在哪里
rpm -qf 命令目录： 查询命令是哪个软件安装过来的
cat :查看文件夹或则文件的内容
chmod +x(权限类型) 文件名：(change modle):修改文件的权限
bsah 文件名： 执行shell 脚本
locate * : 表示查找所有的以某个格式结尾的文件
ln : 创建文件链接 ln -s lijian lijian.sort
ln -s : 创建软链接
alias：查看系统里面已经有了的别名 （临时有效）
alias c=’clear’ 创建别名
unalias: 取消别名
unalias c
source 文件目录 : 将修改的某个文件生效
rmdir(remove directory) : 删除空的文件夹
rm -rf（remove files or directories）(-r:recursive递归删除)(force：强制) ：表示强制删除文件
rm -rf 强制删除的东西不会放在回收站内

1.查看虚拟机的编码：
locale
echo $LANG
2. echo：是查看后面的内容
[root@localhost ~]# echo people
people

  1. $ : 表示某个变量的值
    [root@localhost ~]# shuaige=lijian
    [root@localhost ~]# echo $shuaige
    lijian

4.pwd: 查看当前在那个路径

5.快照：创建一个linux系统好的状态，一旦系统出了问题的时候，可以恢复到当时创建快照时好的系统状态
挂起： 将虚拟机暂停在某个时间点

6.Linux操作系统过程
Linux内核 系统基本库 应用程序

7.[root@localhost ~]# ： 第一提示符PS1
root : 用户 localhost ： 主机名 ~:所处的目录（主目录）
/: 表示根目录 粗体代表文件夹 细体代表文件

  1. echo 与 $ 的使用 $接变量名就是引用变量的值
    [root@li lijian]# shuaige=lijian
    [root@li lijian]# echo $shuaige
    lijian
  1. 修改PS1的值：
    PS1 = ‘[\h@\u \w]$’ => [lijian@root ~]#

10.永久性的修改主机名
进入 /etc/hostname 目录 修改主机名
然后reboot重新启动系统
只有root用户才能改主机名

11.Linux 命令的通用命令格式
命令字符 [选项] [参数]
选项： 段格式 -l 没有接参数的时候 就是默认当前文件夹为参数
长格式 –long format 选项可以组合使用 -al

12.安装使用Tab键补齐命令
yum install bash-conpletion

13.shell的作用：
1. 识别我们输入的命令
2. 帮助执行
所有的可执行文件都放在 /bin 目录下

  1. 基本的硬件：
    CPU 内存 硬盘 网卡 显卡 声卡 输入输出设备
    Linux => 人->shell->系统调用->内核->硬件
    1字节 = 8位
    shell 是解释程序的总称
    Linux 内部命令：属于Shell解释器的一部分，shell自带的命令，不要安装的命令
    外部命令: 独立于shell解释器之外的程序文件，需要额外的安装命令

15.Linux下的快捷键
Tab 自动补齐
Ctrl+C 终止当前进程
Ctrl+L 清屏
Ctrl+K 删除从光标到行末的所有字符
Ctrl+U 删除从光标到行首的所有字符

16.根据不同颜色来判断是什么文件
黑色：代表普通文件
蓝色：代表文件夹
红色：代表压缩文件
绿色： 代表可执行文件
粉色：代表图片

17.-rw-r–r–. 2 root root 20 3月 6 17:14 caiaihui.txt
第一个字符的表示意义 -
- 代表普通文件
d 代表文件夹
l 代表链接文件
c 代表字符设备文件
b 代表块设备文件
