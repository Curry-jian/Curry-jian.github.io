  1. 迭代器：
    迭代器是有状态的，可以被next()调用，函数调用并不断返回下一个值的对象=>迭代器
    x = [1,2,3]
    y = iter(x) => y = x.iter() 可以在可迭代对象中相互转换
    print(next(y)) => 1 next表示一次取出迭代器里面的元素一次取一个元素

7.可迭代对象与迭代器的区别：

取值： index/key 随机取值 只能通过next方法进行向下取值

存储： 一次性占用内存(适合数据量小的时候使用) 只存储下一次取值的计算方法（惰性取值）

迭代器的好处： 在瞬时间 占用的内存少 计算量小

13.用类创建迭代器：
利用菲不拉契数列例子：
class Fib(object):
def init(self):
self.prev = 0
self.current = 1

def __next__(self):
    self.prev,self.current = self.current,self.prev + self.current
    return self.prev
# 创建迭代器  只需要放回他自己就好
def __iter__(self):
    return self

fib = Fib()
print(fib.__next__())

20 * 实例方法： def init(): 这个方法是给实例用的 一定要传递一个实例或者调用
* 静态方法： @staticmethod
* 类方法： @classmethod cls表示类本身 在哪个类里面就表示哪个类 

在方法中，没有使用实例信息，也没有使用类信息     @staticmethod 静态方法
在方法中，没有使用实例信息，使用到了类信息       @classmathod  类方法
在方法中，用到实例信息，没有使用到类信息         普通实例方法
在方法中，用到实例信息和类信息                  实例方法+类方法

例子：

    class Tst(object):
    #类属性(静态属性)
    class_attribute = 'A'

    def __init__(self):
        #实例属性
        self.instance_attrubute = 'B'
    #实例方法
    def instance_method(self):
        print('这是一个实例方法')
        print(self.instance_attrubute)
        print(self.class_attribute)

    # 类方法    =》 装饰器
    #把classmethod下面的方法变为类方法
    @classmethod
    def class_method(cls):
        print('这是个类方法')
        #类方法只能调用类属性
        print(cls.class_attribute)

    #静态方法
    @staticmethod
    def static_method():
        print('这是个静态方法')
        print(Tst.class_attribute)

    #用实例来调用实例方法
    tst = Tst()
    tst.instance_method()

    #用类来调用类方法
    Tst.class_method()
    #用实例来调用方法
    tst.class_method()

    #用实例来调用静态方法
    tst.static_method()
    #用类来调用静态方法
    Tst.static_method()

21.属性包装 ：@ property 将方法转化为属性来使用
当生成一个包装后 python会自动生成 @类名.setter @类名.delete 等包装
例子：

    class Person(object):
def __init__(self,name,age):
    self.name = name
    self._age = age

@property
def age(self):
    print('property_age')
    return self._age

# 这是用做修改的功能
@age.setter
def age(self,value):
      if 0<value<100:
        self._age = value
    else:
        raise ValueError('超出范围')

# 这是删除的功能
@age.deleter
def age(self):
    del self._age

p1 =  Person('cali',19)
#当作属性可以直接访问
print(p1.age)
p1.age = 10
print(p1.age)
